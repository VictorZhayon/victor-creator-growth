## What is the Messages API?

The Anthropic Messages API lets you send structured prompts to Claude and receive intelligent responses. Use it to power chatbots, content generation, data analysis, and more.

## Quick links

- **Get an API key** — Sign up at the [Anthropic Console](https://console.anthropic.com)
- **Install the SDK** — Run `npm install @anthropic-ai/sdk` to get started with Node.js
- **Base URL** — [Anthropic API](https://api.anthropic.com)
